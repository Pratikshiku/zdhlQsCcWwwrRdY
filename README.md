## Agricultural Technology Equipment Rental System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/2084584d867d4c38b6196bb9b51dd7b6/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Announcements
6. Message Feedback
7. Equipment
8. Personal Center
9. Delivery Address
10. Equipment Favorites
11. Equipment Orders


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Post Type Management
5. Announcement Type Management
6. Equipment Type Management
7. Forum Management
8. Announcement Management
9. Message Feedback Management
10. Equipment Management
11. Equipment Evaluation Management
12. Equipment Favorites Management
13. Equipment Order Management
14. User Management
15. Carousel Management
